# Software Engineering - Crypto Spot


## Installation


```bash
git clone
```

## Usage
Go to backend and run these commands

```
cd backend
pip install -r requirement.txt
uvicorn main:app --reload
```

open another terminal window and cd into frontend and run
```
cd frontend
npm start
```
