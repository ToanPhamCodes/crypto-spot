# Software Engineeting - Crypto Spot

Foobar is a Python library for dealing with word pluralization.

## Installation


```bash
git clone
```

## Usage
Go to backend folder and run

```
npm init
```

open another terminal window and cd into frontend and run
```
npm start
```
